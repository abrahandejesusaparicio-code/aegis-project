// Signal Chain

How the T-Connection Works

T-Connection · Ground Relay Node
A

Raven broadcasts over terrain

As Ravens fly their patrol routes they transmit LoRaWAN signals downward and outward. When terrain — hills, buildings, valleys — blocks the direct line to the Principium, the nearest T-Connection node picks up the signal.

B

Node relays to next node

The T-Connection node amplifies and re-broadcasts the signal to the next node in the chain. Each jump covers up to 15 km — nodes can be chained indefinitely across San Miguelito's hills and rooftops.

C

Principium receives everything

Scan maps, waste hotspot data, Havoc telemetry, and route commands all flow through the T-Connection chain back to the Principium HMI — in real time, from any corner of the district.

D

Self-healing if a node fails

The mesh protocol automatically reroutes data around any failed node — signal finds a new path through the network without any manual intervention or downtime.
